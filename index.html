这是仓库根目录的占位页

若你在浏览器里打开本页，说明 GitHub Pages 当前从 main 分支根目录发布，且尚未用下面的 GitHub Actions 或 gh-pages 发布真正的 H5。

请任选一种方式：

  1. 打开仓库 Settings → Pages，Build and deployment → Source 选 GitHub Actions，保存后推送一次 main，等待 Actions 跑完。
  2. 或 Source 选 Deploy from a branch，分支选 gh-pages，目录 / (root)，并在本机 ktv-h5 目录执行 npm run deploy。

正式访问地址应为：https://cisos.github.io/Gyyule/（含末尾路径）。
